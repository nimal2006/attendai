ESP32 Integration Guide

1
ESP32-CAM Setup

Flash the ESP32-CAM with the CameraWebServer sketch. Connect it to your WiFi network. Enter its IP:PORT in the ESP32 Camera tab.

2
ESP32 LCD Setup

Flash the ESP32 with the provided Arduino sketch. It will poll GET /api/hw/esp32/attendance every 5 seconds to receive the latest attendance entry.

3
Data Flow

When attendance is marked (face or manual), the server stores the record. The ESP32 LCD polls the API and displays: Name: John | Status: Present on the LCD.
